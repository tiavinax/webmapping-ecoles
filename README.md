Analyse du besoin
Contexte : Une application webmapping recensant les écoles à Andoharanofotsy (quartier que tu connais bien — c'est sur le trajet du taxibe 137 !).
Utilisateurs cibles :

Parents cherchant une école pour leurs enfants
Habitants du quartier
Administrateurs scolaires


Fonctionnalités proposées
Carte interactive

Afficher toutes les écoles sur la carte avec markers
Popup sur chaque école avec ses informations
Différencier les types d'écoles par couleur de marker (primaire, collège, lycée, privé, public)

Fiche école

Nom de l'école
Type (primaire / collège / lycée)
Statut (public / privé)
Adresse / fokontany
Nombre d'élèves
Contacts (téléphone, email)
Photo

Recherche & Filtres

Rechercher une école par nom
Filtrer par type (primaire, collège, lycée)
Filtrer par statut (public, privé)

Calcul d'itinéraire

Depuis ma position → école choisie (OSRM comme l'exo 4)

Administration (CRUD)

Ajouter une école en cliquant sur la carte
Modifier les informations d'une école
Supprimer une école
